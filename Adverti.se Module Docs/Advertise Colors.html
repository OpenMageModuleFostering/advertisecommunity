Adverti.se Colors

Thank you for installing our Colors module.

Using this this module you can harness image recognition technology to automatically populate the color attribute of your 'simple' products.

For 'configurable' products that are available in a range of colors you can use our color-selector dropdown. It has a wide range of common colors, and color names can be configured to anything you like.

Automatic Color Labelling

Step 1

If you have not done so already, register your Adverti.se account by saving your details in the Magento Admin panel at:

System → Configuration → Adverti.se → Account Settings

At the same time your product image data will be sent to us for processing.

Step 2

You will receive an e-mail to notify you when processing of your image files is complete. Once you receive this e-mail go to the Adverti.se Colors page in your Magento Admin:

Catalog → Adverti.se → Colors

First click the "Get Adverti.se Color Data" button. This will retrieve the color data from our server and save it ready to import into your store.
When the data transfer is complete the grey "Import Colors To Magento" button will become activated and turn orange. Click this button to update the products in your database with the new color values.

Color Selector Drop-Down

For configurable products most Magento themes only allow text-based drop-downs for customers to select the options they want. When you install our Colors module you can let customers see the color they are selecting with our drop-down.

When installed the Adverti.se Colors module adds a new configurable attribute advertise_colors. When products are made configurable on the advertise_colors attribute the Adverti.se Colors drop-down will be used for product option selection in the product pages of your store.

On most installations the Adverti.se Colors drop-down will automatically appear for any configurable products selectable on the advertise_colors attribute. However, if your current theme overrides the default Magento handling of configurable products you may need to make a simple edit to one theme file for the Colors drop-down to work.

To check if you will need to modify a file, check to see if the following template file configurable.phtml exists in your Magento installation:

<Magento-root-folder>/app/design/frontend/<theme-package>/<theme-name>/template/catalog/product/view/type/options/configurable.phtml

If this file does not exist then it means that your theme uses Magento's default template and there is nothing to do - the Adverti.se Colors drop-down should work automatically.

If this file does exist then you need to copy and paste a section of code into it from this file, which was created as part of your Adverti.se Colors installation:

<Magento-root-folder>/app/design/frontend/base/default/template/importer/configurable.phtml

Open this file and copy the section of code between the lines shown: 

  
      <?php /* -+-+-+-+-+-+-+- Start of Adverti.se Color Drop-Down code -+-+-+-+-+-+-+- */ ?>
          ...
          ...
          ...
      <?php /* -+-+-+-+-+-+-+- End of Adverti.se Color Drop-Down code -+-+-+-+-+-+-+- */ ?>

And paste it into your theme's configurable.phtml file immediately below the line:

  
      <?php foreach($_attributes as $_attribute): ?>

You will also need to add a <?php endif;?> immediately before the <?php endforeach; ?>.
